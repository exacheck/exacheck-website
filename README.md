# ExaCheck - Website

This repository contains the GitHub pages source for the [ExaCheck website](https://exacheck.net).

## Development Environment

A VS Code development container is included for easier development.

To run the site inside the development container to preview changes, use these steps:

- Install the requirements

```bash
bundle install
```

- Build the site and start the web server:

```bash
bundle exec jekyll serve
```

The website should then be accessible at [http://localhost:4000](http://localhost:4000).
